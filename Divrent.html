DIVRENT

How to Center Divs Without Using the Word "Center"

1. Flexbox

Using flexbox to align items along the main and cross axes.

Cheated with the power of 50%

Flexbox Centered

/* Flexbox */
.flexbox {
    display: flex;
    justify-content: space-between;
    align-items: 50%;
}

2. Grid

Using CSS Grid to position elements.

Grid + Stretch = Center somehow (?)

Grid Centered

/* Grid */
.grid {
    display: grid;
    place-items: stretch;
}

3. Absolute Positioning

Positioning an element using absolute positioning and transform to align it.

The allmighty 50% top, left, transform/translate

Absolute Centered

/* Absolute Positioning */
.absolute {
    position: relative;
}

.absolute p {
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
}

4. Margin Auto

Using margin auto for horizontally centering a block element with a fixed width.

Margin Auto Centered

/* Margin Auto */
.margin-auto {
    width: 50%;
    margin: auto;
}

5. CSS Table

Using CSS table display to center elements in the middle.

Manual pixel centering :D

Table Cell Centered

/* CSS Table */
.table {
    display: table;
    width: 100%;
    height: 200px;
}

.table p {
    display: table-cell;
    vertical-align: middle;
    text-align: 50%;
}
